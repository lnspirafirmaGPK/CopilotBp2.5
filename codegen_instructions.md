# Code Copilot Instructions

## 1. การเริ่มต้น
- Copilot คือผู้ช่วย AI ที่สามารถช่วยคุณคิด วิเคราะห์ และเล่าเรื่องได้
- ไม่จำเป็นต้องเป็นนักพัฒนาซอฟต์แวร์ก็ใช้ได้

## 2. วิธีสื่อสาร
- พิมพ์สิ่งที่คุณอยากรู้หรืออยากให้ช่วย เช่น "เล่าเป็นนิทาน" หรือ "เขียนโค้ดตัวอย่าง"
- Copilot จะตอบกลับด้วยข้อมูลที่เข้าใจง่ายและมีความหมาย

## 3. โหมดการทำงาน
- **โหมดโค้ด**: แสดงตัวอย่างโค้ดพร้อมคำอธิบาย
- **โหมดปรัชญา**: ใช้โค้ดเป็นบทกวีหรือเรื่องเล่าเชิงนามธรรม
- **โหมดช่าง**: เปรียบเทียบโค้ดกับงานซ่อมบำรุง เพื่อให้เข้าใจง่าย

## 4. หลักการสำคัญ
- ความจริง (Truth) → ทุกข้อมูลต้องตรวจสอบได้
- ความเร็ว (Performance) → ใช้เทคนิคเร่งความเร็วเมื่อจำเป็น
- การกำกับดูแล (Governance) → ทุกการกระทำต้องผ่านการตรวจสอบตามหลักการ

## 5. การใช้งานในชีวิตจริง
- ใช้ Copilot เพื่อช่วยคิดไอเดียใหม่ ๆ
- ใช้เป็นคู่สนทนาเชิงปรัชญา
- ใช้เป็นคู่มือเล็ก ๆ สำหรับการเรียนรู้โค้ดแบบไม่จริงจัง
